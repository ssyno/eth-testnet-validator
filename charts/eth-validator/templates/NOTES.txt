To check the status of your deployment:

  kubectl get pods -n {{ .Release.Namespace }} -l "app.kubernetes.io/instance={{ .Release.Name }}"

To view logs:

  # Geth logs
  kubectl logs -n {{ .Release.Namespace }} -l "app.kubernetes.io/component=execution" -f

  # Beacon logs
  kubectl logs -n {{ .Release.Namespace }} -l "app.kubernetes.io/component=consensus" -f

  # Validator logs
  kubectl logs -n {{ .Release.Namespace }} -l "app.kubernetes.io/component=validator" -f

To check sync status:

  # Geth sync status
  kubectl exec -n {{ .Release.Namespace }} {{ include "eth-validator.geth.fullname" . }}-0 -- \
    geth attach --exec "eth.syncing" /data/geth.ipc

  # Beacon sync status
  kubectl exec -n {{ .Release.Namespace }} {{ include "eth-validator.beacon.fullname" . }}-0 -- \
    curl -s http://localhost:{{ .Values.lighthouse.beacon.ports.http }}/eth/v1/node/syncing

IMPORTANT NOTES:

1. Make sure you have created the required secrets:
   - JWT Secret: {{ .Values.jwtSecret.existingSecretName }} (key: {{ .Values.jwtSecret.key }})
